# AI 憲章 vs. 國家機器 — Anthropic 與川普政府的全面對抗事件簿（2025–2026）

一個**純 HTML、沉浸式、捲動互動的時間軸新聞專題**。以由上至下的時間軸，搭配每則事件的背景圖、交叉淡入轉場、權力天平與權威報導引用，呈現 Anthropic 與川普政府自 2025 年底至 2026 年 6 月的衝突脈絡。

## 🔗 線上瀏覽

👉 **https://austin-liang.github.io/ai-charter-vs-state/**

## ✨ 功能

- 由上至下垂直時間軸，左右交錯卡片（手機自動單欄）
- 兩種閱讀模式：**連續捲動 / 逐頁全螢幕（snap）**，可記憶選擇
- 每則事件配上相關背景圖（取自 Wikimedia Commons），**交叉淡入 + Ken-Burns** 轉場
- 捲動互動：頂部進度條、中央軸線填充、節點發光、權力天平、章節導覽、鍵盤翻頁
- 每個事件節點附**權威報導／官方文件連結**

## ⚠️ 免責聲明

本專題為依據事件簿文件製作的互動式時間軸彙整，整合多家媒體與官方文件之公開報導。事件牽涉政治、法律與企業攻防，發展快速且高度爭議，部分情節具推測性（已於頁面明確標示）。背景圖取自 Wikimedia Commons（公有領域／創用 CC 授權），為相關主體之示意影像，非特定報導當下的新聞照片。
